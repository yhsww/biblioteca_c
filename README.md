# Biblioteca Poe

Projeto acadêmico desenvolvido em equipe para a disciplina de Programação.

## Funcionalidades

- Cadastro de livros
- Cadastro de usuários
- Empréstimos
- Devoluções
- Relatórios
- Persistência em arquivos

## Tecnologias

- Linguagem C
- Manipulação de arquivos
- Modularização
- Git
